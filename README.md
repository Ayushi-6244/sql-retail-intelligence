# 🛒 SQL Retail Intelligence Engine

**Advanced SQL analytics project** simulating a real-world retail business database.
Demonstrates: Window Functions, CTEs, RFM Segmentation, Cohort Analysis & Churn Prediction.

---

## 📌 Business Questions Answered
- Who are our most valuable customers? (RFM Analysis)
- Is revenue growing month-over-month?
- Which products drive the most revenue per category?
- What % of customers from each cohort return after 3 months?
- Which customers are at risk of churning?

---

## 🛠 Tools Used
- MySQL / PostgreSQL
- SQL (CTEs, Window Functions, Aggregations, CASE logic)
- Excel / Tableau (for visualization — optional)

## 🗂 How to Run
1. Run `data/schema.sql` to create tables
2. Run `data/seed_data.sql` to load data
3. Run scripts in `analysis/` folder in order
